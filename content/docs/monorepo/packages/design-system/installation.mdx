[Nextjs](https://nextjs.org/docs) is a React framework for building full-stack web applications. You use React Components to build user interfaces, and [Nextjs](https://nextjs.org/docs) for additional features and optimizations.

## Version Control

Create new Branch(version).

```sh
git checkout -b 05-design-system
```

## Packages Directory

Create a folder called `packages` in the project root.

```sh
cd packages
```

## Install Nextjs

<Steps>
  <Step>
    ### Install

    ```sh
    pnpm create next-app@latest
    ```

  </Step>
  <Step>
    ### Initial Settings

    ```sh
    √ What is your project named? ... ds
    √ Would you like to use the recommended Next.js defaults? » No, customize settings
    √ Would you like to use TypeScript? ... Yes
    √ Which linter would you like to use? » None
    √ Would you like to use React Compiler? ... Yes
    √ Would you like to use Tailwind CSS? ... Yes
    √ Would you like your code inside a `src/` directory? ... Yes
    √ Would you like to use App Router? (recommended) ... Yes
    √ Would you like to use Turbopack? (recommended) ... Yes
    √ Would you like to customize the import alias (`@/*` by default)? ... No
    ```

  </Step>
</Steps>

## Post-Installation

### Delete Folder

- `packages/ds/node_modules`.
- `packages/ds/public`.
- `packages/ds/src`.

## Run

<Steps>
  <Step>
    Back To Root Directory.

    ```sh
    cd ../
    ```

  </Step>
  <Step>
    ### Run Taze

    To ensure all dependencies are up to date and correctly installed, run the following command:

    ```sh
    pnpm dlx taze -r -w
    ```

  </Step>
  <Step>
    ### Run Install

    ```sh
    pnpm i
    ```

  </Step>
  <Step>
    ### Run Dev

    ```sh
    pnpm dev
    ```

  </Step>
</Steps>
