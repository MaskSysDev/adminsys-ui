[Nextjs](https://nextjs.org/docs) is a React framework for building full-stack web applications. You use React Components to build user interfaces, and [Nextjs](https://nextjs.org/docs) for additional features and optimizations.

## Version Control

Create new Branch(version).

```sh
git checkout -b 04-install-nextjs
```

## Apps Directory

Create a folder called `apps` in the project root.

```sh
cd apps
```

## Install Nextjs

<Steps>
  <Step>


  <Step>
    ### Lib Fonts

    Create a folder called `lib` in the `src` directory and inside it create a file called `fonts.ts`. Use the `next/font` package to import and configure the fonts. This allows you to use these fonts throughout your application with optimized loading.

    ```ts title="apps/nextjs16-started/src/lib/fonts.ts"
    import { Geist, Geist_Mono } from "next/font/google"

    export const geistSans = Geist({
      variable: "--font-geist-sans",
      subsets: ["latin"],
    })

    export const geistMono = Geist_Mono({
      variable: "--font-geist-mono",
      subsets: ["latin"],
    })

    ```

  </Step>
  <Step>
    ### Globals CSS

    Create a folder called `styles` in the `src` directory and inside it create the file `globals.css`.

    ```tsx title="apps/nextjs16-started/src/app/layout.tsx"

    import { geistMono, geistSans } from "@/lib/fonts";
    import "@/styles/globals.css";

    export const metadata: Metadata = {
      title: "Create Next App",
      description: "Generated by create next app",

    export default function RootLayout({
      children,

      children: React.ReactNode;
    ```

## Run

<Steps>
  <Step>
    Back To Root Directory.

    ```sh
    cd ../
    ```

  </Step>
  <Step>
    ### Run Taze

    To ensure all dependencies are up to date and correctly installed, run the following command:

    ```sh
    pnpm dlx taze -r -w
    ```

  </Step>
  <Step>
    ### Run Install

    ```sh
    pnpm i
    ```

  </Step>

    ### Run Dev

    ```sh
    pnpm dev
    ```
