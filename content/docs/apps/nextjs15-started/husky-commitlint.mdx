## Version Control

<Steps>
  <Step>
    Send all data to stage.

    ```sh
    git add .
    ```

  </Step>
  <Step>
    Copy data with Commit.

    ```sh
    git commit -m "chore: 04-husky-commitlint"
    ```

  </Step>
  <Step>
    Send Branch.

    ```sh
    git push --set-upstream origin 04-husky-commitlint
    ```

  </Step>
</Steps>
