## Install

<Steps>
  <Step>

    - Ultracite Init

    ```package-install
    npx ultracite init
    ```

    ```sh
    Which editors do you want to configure (recommended)?
    │  VSCode / Cursor / Windsurf
    │
    ◇  Which editor rules do you want to enable (optional)?
    │  GitHub Copilot (VSCode), Cursor, Gemini CLI
    │
    ◇  Would you like any of the following (optional)?
    │  Husky pre-commit hook, Lint-staged
    ```

  </Step>
</Steps>

## Run

<Steps>
  <Step>
    Run Install.

    ```sh
    pnpm i
    ```

  </Step>
  <Step>
    Run Format.

    ```sh
    pnpm format
    ```

  </Step>
  <Step>
    Run Lint.

    ```sh
    pnpm lint
    ```

  </Step>
  <Step>
    Run Check.

    ```sh
    pnpm check
    ```

  </Step>
</Steps>

## Version Control

<Steps>
  <Step>
    Send all data to stage.

    ```sh
    git add .
    ```

  </Step>
  <Step>
    Copy data with Commit.

    ```sh
    git commit -m "feat: 03-biome"
    ```

  </Step>
  <Step>
    Send Branch.

    ```sh
    git push --set-upstream origin 03-biome
    ```

  </Step>
</Steps>
